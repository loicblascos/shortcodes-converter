= Shortcodes Converter =

Convert shortcodes to Gutenberg blocks.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/shortcodes-converter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Tools screen to configure the plugin.
